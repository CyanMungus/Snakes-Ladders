# 🐍 Snakes & Ladders

An online multiplayer Snakes & Ladders game made in HTML, CSS, and JavaScript.

## Features

- 🎲 Dice rolling
- 🐍 Snakes
- 🪜 Ladders
- 👥 1–4 players
- 🌐 Online rooms (coming soon)
- ⚙️ Settings
